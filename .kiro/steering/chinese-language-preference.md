# 中文语言偏好配置

## 语言使用规范

请始终使用中文进行以下操作：

### 1. 对话和回复
- 所有与用户的对话都使用中文
- 技术解释和建议使用中文表达
- 错误信息和状态更新使用中文

### 2. 文档撰写
- 需求文档使用中文
- 设计文档使用中文  
- 任务列表使用中文
- API文档使用中文
- 用户手册使用中文

### 3. 代码注释
- 所有代码注释使用中文
- 函数和类的文档字符串使用中文
- 变量名可以使用英文，但注释必须是中文
- 错误消息和日志输出使用中文

### 4. 命名约定
- 文件名可以使用英文（遵循技术规范）
- 变量名和函数名使用英文（遵循编程规范）
- 但所有相关的注释和文档说明使用中文

## 示例

### 代码注释示例
```typescript
// 上传PDF文件到服务器
async function uploadPdfFile(file: File): Promise<UploadResult> {
    // 验证文件大小不超过1000MB
    if (file.size > 1000 * 1024 * 1024) {
        throw new Error('文件大小超过限制');
    }
    
    // 开始分块上传
    return await chunkUpload(file);
}
```

### 文档示例
```markdown
## PDF预览组件

该组件负责在浏览器中显示PDF内容，支持以下功能：
- 页面缩放和导航
- 缩略图显示
- 文本选择和复制
```

## 技术术语处理

对于技术术语，采用以下原则：
- 优先使用中文术语
- 如果中文术语不够准确，可以使用"中文术语（English Term）"的格式
- 代码中的技术关键词保持英文（如 async、await、Promise 等）

这个配置确保了整个项目的中文化一致性，提高了中文用户的使用体验。
